# Task Management API

Prueba tecnica

### 1. Instalación
```bash
npm install
```

### 2. Configuración
```bash
# Copiar variables de entorno
cp .env.example .env

# Editar .env con tus datos de PostgreSQL
```

### 3. Base de Datos
```bash
# Crear BD y ejecutar migraciones
npm run migrate

# Opcional: datos de ejemplo
npm run seed
```

### 4. Ejecutar
```bash
npm run dev
